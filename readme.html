Getting Started: A 5-Minute Installation

Starting with Random Cats is quick and easy:

  1. Download the package and unzip it into an empty directory.
  2. Open the install.php file in your browser. It will guide you through setting up a wp-config.php file with your database connection details.
  3. If, for any reason, this doesn't work, don't worry. You can manually configure the wp-config.php file by copying the details from wp-config-sample.php.
  4. Once your configuration is set up, the installer will create the necessary tables for your site. If any errors occur, please visit the WordPress support forums with as much information as possible.
  5. After setup, the installer will take you to the login page. You can sign in using the username and password you provided during installation. If a password was generated for you, you can change it in your profile settings.

Keeping Random Cats Up-to-Date

Keeping Random Cats up to date is a breeze:

Using the Automatic Updater

  1. Open the update-core.php file in your browser and follow the instructions.
  2. It's that simple!

Manual Updates

If you prefer a manual update:

  1. Before updating, ensure you have backup copies of any modified files, such as index.php.
  2. Delete your old Random Cats files, but keep any modifications you've made.
  3. Upload the new files.
  4. Visit /wp-admin/upgrade.php in your browser to complete the update.

Migrating from Other Systems

You can easily migrate content from various systems to Random Cats. Just ensure that Random Cats is installed and functioning before using our import tools.

System Requirements

To run Random Cats, make sure you have the following:

  • PHP version 7.0 or higher.
  • MySQL version 5.0 or higher.

Recommended Setup

For the best experience, we recommend the following:

  • PHP version 7.4 or higher.
  • MySQL version 5.7 or higher OR MariaDB version 10.4 or higher.
  • The mod_rewrite Apache module.
  • HTTPS support.
  • Adding a link to wordpress.org on your site is appreciated.
